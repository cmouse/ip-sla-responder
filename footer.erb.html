IP SLA responder

This software is intended as drop-in replacement for Cisco or Juniper device for IP SLA measurement counterpart. It acts as responder for measurement messages sent by network devices. It lets you configure one IP address that will be responded to on up to 4096 different VLANs without configuring these or the IP address on the server.
